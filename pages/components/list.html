List

To create lists use "list" class, additionally use "list-roman" or "list-alpha" class to create desired components.

Notification Drawer

Create a notification drawer using the "notification-drawer" class.

Notifications

              <div class="notification-drawer">
                <header>
                  <h1 class="heading">Notifications</h1>
                  <i class="fas fa-ellipsis-v icon"></i>
                </header>
                <div class="notification-divider"></div>
                <section>
                  <ul class="list">
                    <li class="notification">
                      <span class="title">Upload Instagram Profile</span>
                      <p class="subtitle">2 day ago</p>
